CNC Aluminum Bar G-code Generator

Simply enter the number of holes and the file name to automatically generate two .nc files based on your current machining rules: hole and edge.

Machining Note: This version is only intended for aluminum bars with a thickness of 1 mm. The generated hole and edge G-code are both based on this thickness setting.
  • Hole positions start from Y10 automatically, with 20 mm spacing between holes
  • Aluminum bar length is calculated automatically as 20n mm
  • The edge program Y value is calculated automatically as 20n + 2

Input Parameters

Please complete all required fields before clicking Generate G-code.

Must be a positive integer, such as 1, 2, or 10.

Automatic calculation formula: 20 × hole count

Enter only the base file name. Do not add _hole, _edge, or .nc.
